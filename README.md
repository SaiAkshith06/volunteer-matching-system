# Smart Resource Allocation

A data-driven volunteer matching system that helps NGOs assign the right people to the right tasks in seconds.

## Features
- CSV upload for volunteers and needs
- Intelligent matching algorithm
- Top match recommendations
- One-click assignment
- Real-time dashboard
- Export results as CSV

## Tech Stack
- React + TypeScript
- Tailwind CSS
- Papaparse
